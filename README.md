# dadata-suggestions

Попытка портировать [suggestions-jquery](https://github.com/hflabs/suggestions-jquery) на Vue 3

## Project Setup

```sh
bun install
```

### Compile and Hot-Reload for Development

```sh
bun dev
```

### Build for Production

```sh
bun run build
```

### Lint with ESLint

```sh
bun lint
```
