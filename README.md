# dadata-suggestions

Форк библиотеки [suggestions-jquery](https://github.com/hflabs/suggestions-jquery) для автодополнения адресов, организаций, имен и других данных через API DaData.

## Статус проекта

🚧 **В разработке**

Отличия от оригинальной библиотеки:

- Полностью удалён jQuery(в будущем планирую использовать vue).
- Аргументы в событиях suggestions-\* теперь в виде массива.
- Поддержка TypeScript(в процессе).
- Современный синтаксис для современных браузеров.
- Упрощённый код и уменьшенный размер библиотеки.

Что удалено:

- Реклама DaData(promo).
- Работа без CORS.
- Отложенная инициализация.
- Методы disable/enable.
- ФИАС.
- Версия и заголовок(header) X-version.
- scrollOnFocus.

Портированы тесты из оригинального репозитория, за исключением тестов, связанных с удалёнными функциями.
